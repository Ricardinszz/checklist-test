Checklist

GESTÃO E GOVERNANÇA DE TECNOLOGIA DA INFORMAÇÃO

  • Gerenciamento de projetos (PMBOK 7ª edição)
  • Processos, grupos de processos e área de conhecimento
  • Gestão de riscos
  • Gerenciamento de serviços (ITIL v4)
  • Governança de TI (COBIT 2019)
  • Conceitos de gestão de processos e modelagem de processos de negócio usando BPMN
  • Instrução Normativa SGD n.º 1/2019 e suas alterações
  • ISO 20.000 e ISO 19.770
  • Gestão de Indicadores
  • Kanban
  • Lei 13.303/2016 e suas alterações
  • Instrução Normativa SGD nº 05/2021 e suas alterações
  • Instrução Normativa SGD nº 94/2022 e suas alterações

SUPORTE E INFRAESTRUTURA

  • Arquitetura de Rede TCP/IP
  • Arquitetura hardware de servidores
  • Armazenamento de dados
  • Rede SAN (storage area network)
  • Conceitos de armazenamento de discos e conceito de replicação de dados
  • Formatação de dados
  • Virtualização (VMWare e HyperV)
  • Consolidação de servidores
  • Cluster (alta disponibilidade e performance)
  • Conceitos de mensageria
  • Computação em grid e em nuvem
  • Nuvem pública e nuvem privada
  • Conceitos de ambiente bare metal
  • Servidores de aplicação (IIS e APACHE)
  • Noções de servidores de páginas em HTML: Nginx e Apache
  • Conceito de servidores de armazenamento orientado a objetos (object store): S3

BANCO DE DADOS

  • Modelagem de dados (conceitual, lógica e física)
  • Abordagem relacional
  • Normalização das estruturas de dados
  • Integridade referencial
  • Metadados
  • Modelagem dimensional
  • Linguagem de consulta estruturada (SQL)
  • Linguagem de definição de dados (DDL)
  • Linguagem de manipulação de dados (DML)
  • SGBD
  • Propriedades de banco de dados
  • Banco de dados NoSQL
  • Banco de dados em memória
  • Data lakes e soluções para big data
  • Conceitos de Inteligência Artificial, Análise de Dados e Big Data

REDES DE COMPUTADORES

  • Conceitos de redes de computadores: meios de transmissão, classificação, topologia de redes, redes de longa distância, redes locais e redes sem fio
  • Elementos de interconexão de redes de computadores (hubs, repetidores, switches, roteadores). VLANs. Cabeamento estruturado
  • Noções dos modelos de referência OSI (Open System Interconnection Reference Model)
  • Noções dos padrões IEEE 802.1, IEEE 802.3, IEEE 802.11 a/b/g/n/ac
  • Arquitetura e pilhas de protocolos TCP/IP: Camada de rede (IPv4, IPv6 e IPsec), conceitos básicos de endereçamento e roteamento; Camada de transporte (TCP e UDP); Camada de aplicação (FTP, SSH, DNS, SMTP, POP, IMAP, HTTP, HTTPS, SSL, DNS, RDP, DHCP)
